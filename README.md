# 🎓 Edutech Enrollment Platform

A web-based platform that allows students to browse training programs, register, and complete their enrollment with payment tracking.

## 🚀 Features

* 📚 View available training programs
* 📝 Student registration form
* 💳 Payment status tracking (paid / unpaid)
* 🗄️ MySQL database storing user data
* 🔎 Admin can verify registrations (if implemented)

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server: XAMPP (Apache)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/wnsomar6-rgb/edutech-enrollment-platform.git
```

2. Move the project folder to:

```
C:/xampp/htdocs/
```

3. Start XAMPP:

* Start **Apache**
* Start **MySQL**

4. Import the database:

* Open phpMyAdmin (http://localhost/phpmyadmin)
* Create a database
* Import the `.sql` file provided in the project

5. Run the project:

```
http://localhost/edutech-enrollment-platform
```
